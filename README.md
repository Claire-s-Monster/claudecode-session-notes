# Session Notes

A tool for managing development session notes and task tracking.

## Description

Session Notes helps developers maintain organized notes during development sessions, track progress, and manage tasks effectively. This tool integrates with development workflows to provide structured note-taking and task management capabilities.

## Installation

```bash
# Clone the repository
git clone <repository-url>
cd session-notes

# Install with pixi (recommended)
pixi install

# Or install with pip in development mode
pip install -e .
```

## Development

```bash
# Install development dependencies
pixi install

# Run tests
pixi run test

# Run linting
pixi run lint

# Run full quality check
pixi run quality
```

## Features

- Session-based note organization
- Task tracking and progress monitoring
- Development workflow integration
- Structured data storage and retrieval

## Requirements

- Python 3.12+
- Git (for development)
